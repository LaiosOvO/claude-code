# 从基础到全貌 — 自底向上理解 Claude Code Best

!!! info "阅读建议"
    本教程适合希望**从具体代码入手，逐步构建全局理解**的读者。如果你更喜欢先建立宏观认知，可以参考 [自顶向下教程](../top-down/index.md)。

## 阅读路径

本教程基于 **Claude Code Best (ccb)** 的真实代码，从最底层的基础原语逐层上升到完整系统。

```
第一章 → 第二章 → 第三章 → 第四章 → 第五章
基础原语   核心抽象   模块组装   系统集成   全景视角
(砖块)    (零件)    (组件)    (系统)    (全貌)
```

## 章节概览

| 章节 | 标题 | 核心问题 |
|------|------|----------|
| [第一章](01-fundamentals.md) | 基础原语 | Signal、Zod、AsyncGenerator 等底层构建块 |
| [第二章](02-core-abstractions.md) | 核心抽象 | Tool、Command、Skill 如何定义？ |
| [第三章](03-module-assembly.md) | 模块组装 | 58+ 工具如何组成工具池？MCP 如何集成？ |
| [第四章](04-system-integration.md) | 系统集成 | 完整请求从用户输入到结果返回的全链路 |
| [第五章](05-full-picture.md) | 全景视角 | 站在山顶回望，关键设计决策总结 |

## 与自顶向下教程的关系

两套教程覆盖相同的代码，但视角不同：

- **自顶向下**：全局 → 局部，快速建立全景
- **自底向上**：局部 → 全局，深入理解细节

建议先读完一套再选读另一套的感兴趣章节。
